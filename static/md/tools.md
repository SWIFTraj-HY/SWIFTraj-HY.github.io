## Tools & Visualization
<!-- 工具与可视化 -->

We provide python scripts to load and visualize the dataset.

### Prerequisites
- Python 3.8+
- numpy
- pandas

### Installation

```bash
pip install -r requirements.txt
```

### Usage

```python
import dataset_loader
data = dataset_loader.load('path/to/data')
dataset_loader.visualize(data)
```
